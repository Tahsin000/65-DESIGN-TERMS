Lesson - 2 [ 11 important color definitions ]

  1. CMYK

CMYK stands forcyan magenta yellow and k for key orblack these are the four most basiccolors you can use for printing you willnotice cmyk colors aren't as vibrant asrgb and this is because cmyk issubtractive this means that the colorswork with the reflected light

  1. RGB

RGB colorsare used on screen only it stands forred green and blue and this color modeis additive meaning that by mixing theseprimary colors in different combinationswe can simulate a bigger range of colors

  1. Grayscale :

grayscale is a monochromaticpalette using different shades of grayin an image a single image is composedof up to 256 combinations of shades of greywhat is

  1. Opacity :

opacity refers to thelevel of transparency on an element thelower the opacity the more transparentthe element is if the opacity is set to100 percent it means the object is solid

  1. Saturation :

saturation refers to the intensity of a color you will noticesaturated images appear brighter and punchier the saturated images are dulland almost colorless

  1. Hue :

hue is a way to describe a purecolor without tint or shade that meansadded white or black for instance anycolor on the color wheel is a hue blue yellow red

  1. Tone :

tone refers to a hue with added gray thetone will lower the intensity of colorand it can become dull

  1. Tint :

tint refers to a hue with added white to lighten it and make it paler the tint can range anywhere from a slightly lighter color all the way to completely white with barely any color

  1. Shade :

shade refers to black added to a color this is theoppo site of tint instead of making the color lighter shade will darken

  1. Color palette :

color palette a color paletteis a group of colors that can be usedfor a specific design project this color palette usually represents a brand and should be chosen to work in harmony with each

  1. Pantone system :

pantone systempms or pantone matching system is the most widely used system for blending colors that aren't cmyk every hue is identified by a number that is easy for designers to reference and reproduce when printing
